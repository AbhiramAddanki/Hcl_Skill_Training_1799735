There is an integer array nums sorted in ascending order (with distinct values).Prior to being passed to your function, nums is possibly left rotated at an unknown index k (1 <= k < nums.length) such that the resulting array is [nums[k], nums[k+1], ..., nums[n-1], nums[0], nums[1], ..., nums[k-1]] (0-indexed). For example, [0,1,2,4,5,6,7] might be left rotated by 3 indices and become [4,5,6,7,0,1,2].
Given the array nums after the possible rotation and an integer target, return the index of target if it is in nums, or -1 if it is not in nums.You must write an algorithm with O(log n) runtime complexity.

 

Example 1:

Input: nums = [4,5,6,7,0,1,2], target = 0
Output: 4



Approach:
In a rotated sorted array, we cannot directly apply normal binary search because the array is not completely sorted. The rotation breaks the global ordering, so the target may exist on either side even if `nums[mid]` is greater or smaller than the target.

However, an important observation is that at any step of the array, at least one half of the array will always be sorted.

So the approach is:

1. Use binary search and calculate the middle index.
2. If the middle element is equal to the target, return the index.
3. Check which half of the array is sorted.
4. If `nums[left] <= nums[mid]`, then the left half is sorted.
5. Check if the target lies between `nums[left]` and `nums[mid]`.
6. If the target lies in this range, move the search to the left side by updating `right = mid - 1`.
7. Otherwise, move to the other half by updating `left = mid + 1`.
8. If the left half is not sorted, then the right half must be sorted.
9. Check if the target lies between `nums[mid]` and `nums[right]`.
10. If it lies in this range, move to the right side (`left = mid + 1`), otherwise search the left side (`right = mid - 1`).

Repeat this process





code:
class Solution {
public:
    int search(vector<int>& nums, int target) {
        int n=nums.size();
        int low=0;
        int high=n-1;
        while(low<=high){
            int mid =low+(high-low)/2;
            if(nums[mid]==target){
                return mid;
            }
            if(nums[low]<=nums[mid]){
                if(nums[low]<=target&&nums[mid]>target){
                    high=mid-1;
                }
                else{
                    low=mid+1;
                }

            }
            else{
                if(nums[mid]<target&&nums[high]>=target){
                    low=mid+1;
                }
                else{
                    high=mid-1;
                }

            }
        }
        return -1;
    }
};
